# Other Mini Projects

A collection of small projects, experiments, lab activities, integrations, and technical notes that don't require dedicated repositories.

This repository serves as a central place to document practical learning, cybersecurity research, IT administration tasks, automation workflows, AI integrations, and personal lab experiments.

Most projects are intentionally lightweight and documented in a single Markdown file.

## Why This Repository?

Not every project needs its own repository.

Many useful experiments can be explained in a single document while still providing value, documenting lessons learned, and serving as a future reference.

This repository is my personal collection of those projects.
